# 🌐 Personal Portfolio Website

This project is a multi-page personal portfolio website developed as part of a weekly Web Programming course. Each week, new features and improvements were added, resulting in a complete and structured final product.

## 🚀 Live Demo

You can view the live version here:
👉 https://sudenazgrmz.github.io

---

## 📌 Project Purpose

The main goal of this project is to:

* Learn and apply HTML5 and CSS3 fundamentals
* Understand layout techniques such as Flexbox
* Build a structured and responsive multi-page website
* Publish a project using GitHub Pages

---

## 🧩 Features

* Multi-page structure (Home, About, Interests, Contact)
* Responsive layout using Flexbox
* Clean and modern UI design
* Navigation bar with active states
* Styled contact form
* Organized file structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Git & GitHub
* GitHub Pages
